# Git4Gits

Education initiative: help **subject matter experts** ship small, verifiable software with **AI coding tools**—without pretending everyone becomes a career engineer.

## Start here

1. Open [`plan/00-start-here.md`](plan/00-start-here.md) for locked-in v0 defaults and this week’s actions.
2. Run the **Week 1 checklist** in the same folder.
3. Use [`templates/problem-brief-template.md`](templates/problem-brief-template.md) before any codegen in teaching or pilot sessions.

## Repo layout

| Path | Purpose |
|------|---------|
| `plan/` | Decisions, checklists, scripts for discovery and pilot |
| `templates/` | Reusable briefs and review checklists for learners |
