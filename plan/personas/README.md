# Synthetic learner personas (`plan/personas/`)

Synthetic **test doubles** for facilitation dry-runs—not learner “play personas.” Player-facing guild roles live in [`gamification-rewards-and-identity.md`](../gamification-rewards-and-identity.md).

## How to “randomise”

1. **Expert-engineer smoke test:** open **[`persona-07-avi-technical-master.md`](persona-07-avi-technical-master.md)** (#07) plus the linked knowledge reference—skip toolchain 101.
2. **Industry rotation:** Ask someone to roll a die (`1–6`) or pick **`persona-0[1-6]-*.md`** at random—or run `ls plan/personas/persona-0[1-6]-*.md | shuf | head -1` locally.
3. Open that file and treat it as completed Part 1.
4. Run [`templates/learner-intake-followup-generator-prompt.md`](../templates/learner-intake-followup-generator-prompt.md) against **only** anonymised excerpts if you paste into vendor LLMs—some orgs forbid even fictional employer detail.

Do **not** present these bios as research or leaked facts.
