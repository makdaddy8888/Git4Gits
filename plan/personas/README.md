# Synthetic learner personas (`plan/personas/`)

These people **do not exist**. Use them for **design reviews**, **dry-run facilitation**, filling `learner-intake-form.md`, or **testing** “elite vs standard” paths.

## How to “randomise”

1. Ask someone to roll a die or pick **`persona-<n>-*.md`** at random—or run `ls plan/personas/*.md | shuf | head -1` locally.  
2. Open that file and treat it as completed Part 1.  
3. Run [`templates/learner-intake-followup-generator-prompt.md`](../templates/learner-intake-followup-generator-prompt.md) against **only** anonymised excerpts if you paste into vendor LLMs—some orgs forbid even fictional employer detail.

Do **not** present these bios as research or leaked facts.
